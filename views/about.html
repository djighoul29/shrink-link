Shrink Link

About

Shrink Link is a minimalist web service designed for quickly and conveniently shortening long URLs. This project was developed as part of a college assignment, with the main goal being to create a fully functional web application without using external frameworks. The backend is powered by Node.js, handling server-side logic without additional libraries, while the frontend is styled with PicoCSS for a lightweight and responsive design. Vanilla JavaScript manages client-side interactions, and the shortened links are stored using either the file system or a database.

Shrink Link will be part of a larger project – my web platform Coders Cave, a place for developers and PC enthusiasts (currently in development).
